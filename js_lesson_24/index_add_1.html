Додаткове завдання 1

Task 1. З клавіатури поступово вводить користувач числа. Підрахувати кількість різних введедних чисел. Приняє введення тоді, коли введе 0. Task 2. Підрахувати кількість різних імен. Task 3. Дано масив книг (назва, рік видання, автор (ПІБ, рік народження). Підрахувати кількість різних авторів. Task 4. Підрахувати кількість входжень кожного із значень. Task 5. Дано список з віком учнів. Підрахувати скільки разів кожне значення зустрічається у списку і максимальне.
