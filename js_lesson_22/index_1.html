Урок 14 - Домашнє завдання 1 - 9

Задача 1. Визначити привітання у залежності від часу доби (Доброго дня, Доброго ранку, Доброго вечора). Задача 2. Вивести дату у форматі “місяць – рік – вихідний/робочий”. Задача 3. Визначити поточну пору року. Задача 4. Вивести назву дня. Задача 5. Користувач вводить дату народження. Визначити, скільки йому повних років. Задача 6. Користувач іде у відпустку на 56 днів. Визначити дату на момент його повернення. Задача 7. Користувач бере авто на вказану кількість годин. Визначити кінцевий час повернення. Задача 8. Задаємо користувачу задачу з додавання двох чисел. Визначити скільки секунд було використано для розв’язання задачі. Задача 9 . Дано дата виробництва йогурта (вводимо рік, місяць, день) та кількість днів придатності. Визначити чи є він придатним на даний момент.

Task 1. Determine greetings depending on the time of day (Good afternoon, Good morning, Good evening). Task 2. Output the date in the format "month - year - holiday/working day". Task 3. Determine the current season. Task 4. Output the name of the day. Task 5. The user enters the date of birth. Determine how old he is. Task 6. The user goes on vacation for 56 days. Determine the date for his return. Task 7. The user takes the car for the specified number of hours. Determine the final return time. Task 8. We ask the user the problem of adding two numbers. Determine how many seconds were used to solve the problem. Task 9. Date of production of yogurt (enter year, month, day) and the number of expiration days are given. Determine whether it is suitable at the moment.
